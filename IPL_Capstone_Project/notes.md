# IPL Capstone Project Notes

## Project Objective

The goal of this project was to perform Exploratory Data Analysis (EDA) on IPL match data and extract meaningful insights using Pandas, Matplotlib, Seaborn, and Plotly.

---

## Skills Applied

* Data Cleaning
* Data Exploration
* GroupBy Aggregations
* Data Visualization
* Statistical Analysis
* Feature Extraction
* Business Insights Generation

---

## Questions Explored

* Which team won the most matches?
* Which venue hosted the most games?
* Which player won the most Player of the Match awards?
* Toss Winner vs Match Winner Analysis
* Team-wise Performance Analysis
* Season-wise Trends
* Venue Analysis

---

## Libraries Used

* Pandas
* NumPy
* Matplotlib
* Seaborn
* Plotly

---

## Key Learning

This project helped connect data manipulation and visualization concepts learned throughout the Data Visualization journey into a real-world dataset.

The focus was not only on creating charts but also on extracting actionable insights from data.

---

## Outcome

Successfully completed an end-to-end IPL Data Analysis project using industry-standard Python libraries.
